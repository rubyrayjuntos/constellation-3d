## The Color Code (Know This First)

GOSP uses three colors functionally:

- **üîµ CYAN** = Structure (the rigid parts, scaffolds, architecture)
- **üü£ MAGENTA** = Dynamics (the flexible parts, active zones, chaos)
- **üü° AMBER** = Coupling (the connection, the pathway, the focal point)

**When to use which:**
- Left side = usually cyan (structure)
- Right side = usually magenta (dynamics)
- Center = usually amber (where they connect)

**Exception:** If you WANT new properties to emerge (like the Empress creating flowers), expect colors beyond these three. That's good - it means the system added information.

## Evaluating Your Result

Ask yourself:

1. **Constraint Check:**
   - Did it use the right colors in the right places?
   - Did it show the direction you specified (left‚Üíright vs right‚Üíleft)?
   - Did it capture the material properties (solid vs translucent)?

2. **Emergence Check:**
   - Did anything appear you didn't specify?
   - Does it feel like it's showing the actual PROCESS?
   - Does it resonate with the concept?

3. **Transfer Test:**
   - Could you apply the same constraints to a different concept?
   - Would it still make sense?

**If 2/3 checks pass ‚Üí you did GOSP successfully!**

## Common Mistakes

**‚ùå Being too vague:**
"Make a mystical energy thing"
‚Üí System has no constraints, produces generic output

**‚úì Being specific about relationships:**
"Asymmetric gradient, cyan left, magenta right, amber connecting them"
‚Üí System has clear constraints, produces coherent output

---

**‚ùå Describing appearance:**
"It should look beautiful and spiritual"
‚Üí Aesthetic goals don't constrain the solution

**‚úì Describing physics:**
"Chaos ‚Üí order phase transition, entropy decreasing locally"
‚Üí Physical constraints do constrain the solution

---

**‚ùå Forcing every detail:**
"Place a sphere at position (2,3) with radius 1.5 colored #FF00AA"
‚Üí Over-specification makes the system resist

**‚úì Specifying relationships:**
"Magenta chaos on right compresses through amber focal point into cyan structure on left"
‚Üí Relationship specification gives system room to solve
